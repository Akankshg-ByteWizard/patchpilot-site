Roadmap by Milestone

Milestones are designed to deliver usable value early and compound reliability with each release.

Milestone 1: Working Hook + Daemon

Capture failed commands and persist structured sessions locally.

Milestone 2: Reliable Fix Proposals

Generate patch and command recommendations with confidence scores.

Milestone 3: Memory Layer

Add vector retrieval and write-back from successful sessions.

Milestone 4: Agentic Reasoning

Enable tool-using multi-step flows for cascading failure recovery.

Community FAQ

Will this remain open source?

Yes. The initial roadmap and core engine are planned to stay open and community driven.

Can contributors own modules?

Yes. Maintainers can take ownership areas such as shell hooks, agent tools, or memory.

Is local-only mode supported?

Yes. Local inference endpoints and local storage are first-class workflows.
